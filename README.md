# Chess Game by 23NeuroBytes

A real-time multiplayer chess game built using **Socket.io** and **Chess npm modules**. Share your unique game link with an opponent and start playing instantly!

## 🏆 Features
- **Real-time gameplay** powered by WebSockets.
- **Interactive chessboard** with legal move validation.
- **Lightweight & fast**—play directly from your browser.
- **No sign-up required**—just share the link and play!

## 🛠️ Technologies Used
- **Socket.io** – Enables real-time communication.
- **Chess.js** – Handles chess rules and logic.
- **Chessboard.js** – Renders a dynamic chessboard.
- **Node.js** – Backend processing and game state management.
